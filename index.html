Hoş Geldiniz

Bu web sitesi Web Programlama dersi kapsamında hazırlanmış basit bir kişisel tanıtım sayfasıdır. HTML ve CSS kullanarak sayfa düzeni oluşturma çalışmasıdır.

İçerik

Bu bölüm sayfanın ana içerik alanıdır. CSS özellikleri ile metinlerin boyutu, rengi ve satır aralığı düzenlenebilir.

Kutular yan yana yerleştirilerek sayfa düzeni oluşturulabilir ve farklı arka plan renkleri kullanılabilir.

Basit web sitelerinde içerik ve bilgi bölümleri birlikte kullanılır.

Yan Bilgi

Bu bölüm sayfanın sağ tarafında bulunan bilgi alanıdır.

CSS ile genişlik, margin, padding ve border özellikleri kullanılarak tasarım yapılabilir.

Web tasarımında bu yapı oldukça yaygın kullanılmaktadır.
